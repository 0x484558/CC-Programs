# CC Programs

A small collection of Lua programs for ComputerCraft and CC:Tweaked.

Each program is self-contained and can be copied to a ComputerCraft computer or turtle as needed. See the comments at the top of a program for its configuration and usage details.

## Programs

- `quarry.lua` — a turtle quarry program.
